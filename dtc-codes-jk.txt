# JK Wrangler (2007-2018) DTC code reference
# Tab-separated:  <CODE>\t<DESCRIPTION>
# Comments begin with #
#
# Most P0xxx codes are SAE J2012 standardised and apply to other
# FCA / Stellantis platforms (JL Wrangler, JT Gladiator, Grand
# Cherokee, Ram).  P1xxx, Bxxxx, Cxxxx, Uxxxx codes are manufacturer-
# specific to FCA and may not transfer 1:1 to non-FCA vehicles.
#
# Lookup format used by dtc.sh: matches first whitespace-delimited
# token after stripping a leading "#" comment.
#
# Duplicates in the upstream source have been collapsed.  Where
# different descriptions exist for the same code (B1644, B210D,
# B210E, U0121, U0141, U0151, U0155, U0184, U0186), the most
# descriptive variant is kept.
#
# ===========================================================
# BODY  (B-codes)  --  audio, HVAC, lighting, locks, RKE, airbags
# ===========================================================
